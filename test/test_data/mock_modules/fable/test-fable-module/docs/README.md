# Test Fable Module

A test module for validating documentation catalog generation.

## Features

- Dependency injection via Fable service provider pattern
- Configurable logging with multiple output targets
- UUID generation for unique identifiers

## Installation

```bash
npm install test-fable-module
```

## Quick Start

```javascript
const libFable = require('fable');
let tmpFable = new libFable();
tmpFable.log.info('Hello from Fable!');
```
