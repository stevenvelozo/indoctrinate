# Configuration

Configuration options for the test fable module.

## Settings

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| Product | string | "Fable" | Application name |
| LogLevel | number | 3 | Logging verbosity level |
| UUID | boolean | true | Enable UUID generation |

## Environment Variables

You can override settings with environment variables:

- `FABLE_PRODUCT` - Override the product name
- `FABLE_LOG_LEVEL` - Override the log level
